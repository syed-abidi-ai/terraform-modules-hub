# terraform-modules-hub
To store the terraform modules for reproducibility 
```bash
terraform-modules-hub/
├── github/
│   ├── repo-create/
│   ├── branch-protection/
│   └── team-management/
├── aws/
│   ├── vpc/
│   ├── ec2/
│   └── s3/
├── azure/
│   ├── resource-group/
│   ├── vm/
│   └── storage-account/
├── gcp/                      # Optional
│   └── cloud-function/
├── modules.tf                # Optional root module references
├── README.md                 # Overview of module usage
└── examples/                 # Usage examples per provider
```
